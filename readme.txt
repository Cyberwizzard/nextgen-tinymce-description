NextGEN TinyMce Description add native tinymce to nextgen gallery picture description.

== Description ==

NextGEN TinyMce Description simply integrate tinymce wysiwyg to the description of the picture in a NextGEN gallery.

== Installation ==

1. 	Copy the plugin in wp-content/plugins directory

2.	Activate the plugin

3. 	Go to your "manage gallery" on NextGEN, you will see description textarea modified. There is no configuration panel

That's it ... Have fun
